# Django 실행
python manage.py runserver

# 마이그레이션
python manage.py makemigrations
python manage.py migrate

# 블로그의 마이그레이션 추가
python manage.py makemigrations blog
python manage.py migrate

# 슈퍼어드민 생성
python manage.py createsuperuser

# 파일기반 DB : sqlite

# Git
- git add .
- git commit -m "commit massage"
- git push

# settings.py
- INSTALLED_APPS : 만든 앱 목록 -> 블로그나 자기소개 서비스를 만든다면 해당 부분에 추가해주어야겠지
    * run manage.py (python manage.py startapp blog)


static path : 클라이언트가 서버의 자원에 접근 가능하도록 하는 경로
media path : 클라이언트가 업로드하는 위치를 지정
path.join을 사용하는 이유는 OS에 따른 경로 규칙이 다르기 때문에 통일시키기 위함
